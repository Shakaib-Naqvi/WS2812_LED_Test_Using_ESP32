## WS2812 LED Control with ESP32

This project controls a WS2812 (NeoPixel) LED strip using an ESP32 and the Adafruit NeoPixel library. It demonstrates multiple LED effects including color wipe, theater chase, and rainbow animations.

### Hardware

- ESP32
- WS2812 / WS2812B LED strip (50 LEDs)
- External 5V power supply (recommended)

#### Data Pin: GPIO 12
#### Baud Rate: 115200

### Features

- Color wipe (Red, Green, Blue)

- Theater chase effects

- Rainbow & rainbow cycle animations

- Adjustable brightness

- Library

- Adafruit NeoPixel

### Usage

Upload the code to ESP32, power the LED strip, and view logs on the Serial Monitor.
